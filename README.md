# 加工ツール集 / Machining Tools

切削加工の計算をブラウザ上で手軽に行えるWebツール集です。
インストール不要。HTMLファイルをブラウザで開くだけで使えます。

---

## ツール一覧

### 1. 加工時間計算ツール `machining_time_calculator.html`

長さと送り速度から加工時間を計算します。

| 入力 | 単位 |
|------|------|
| 加工長さ L | mm |
| 送り速度 F | mm/min |

| 出力 | 単位 |
|------|------|
| 加工時間 T | 分 / 秒 / ミリ秒（自動切替） |

**計算式**
```
T = L / F
```

---

### 2. 切削速度 / 回転数 変換ツール `cutting_speed_calculator.html`

切削速度と回転数を相互に変換します。モードを切り替えて使用します。

**モード1：切削速度 → 回転数**

| 入力 | 単位 |
|------|------|
| 切削速度 Vc | m/min |
| 工具径 D | mm |

| 出力 | 単位 |
|------|------|
| 回転数 n | min⁻¹ |

```
n = (Vc × 1000) / (π × D)
```

**モード2：回転数 → 切削速度**

| 入力 | 単位 |
|------|------|
| 回転数 n | min⁻¹ |
| 工具径 D | mm |

| 出力 | 単位 |
|------|------|
| 切削速度 Vc | m/min |

```
Vc = (π × D × n) / 1000
```

---

## 使い方

1. HTMLファイルをブラウザで開く（またはGitHub PagesのURLにアクセス）
2. 数値を入力すると自動で計算される
3. リセットボタンで入力をクリア

## 動作環境

ブラウザのみ。サーバー・インストール不要。
Chrome / Firefox / Safari / Edge に対応。
